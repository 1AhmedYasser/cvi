# Font size scale

See [information on typography on the Veera Design System page](https://veera.eesti.ee/3d136290e/p/070c6e-tpograafia/b/1507db). 

Veera has many font sizes to choose from. 

You can pick a font size by employing Sass variables (prefixed with `$veera-font-size`).
