## Styles

<Story name="Styles library">
  {{

      Latest version: {{ stylesPkg.version }}
    `,
  }}
</Story>

### Installation

1. Install the library
```
npm i --save @ria/veera-styles
```

2. Include styles in the project

`@use` or `@forward` the styles into your stylesheet (install Dart Sass):

```
@use '~@ria/veera-styles';
```

```
"@angular/platform-browser": "^14.1.1",
"@angular/platform-browser-dynamic": "^14.1.1"
```

### Installation

1. Install the library
```
npm i --save @ria/veera-ng;
```

2. Import UiModule in your app.module.ts.

```
import { UiModule } from '@ria/veera-ng';

@NgModule({
  ...
  imports: [BrowserModule, UiModule],
  ...
})
```
