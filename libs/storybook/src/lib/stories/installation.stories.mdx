# Installation

Installation is as easy as a pie:

1. Install the project
```
npm install '@ria/veera-styles';
```

2. `@use` or `@forward` the styles into your stylesheet (make sure Dart Sass is installed):
```
@use '@ria/veera-styles'
```

3. Done, enjoy!
