## General description

| Feature                    | Description       |
|----------------------------|-------------------|
| Selector                   | `veeraNgTooltip`  |
| Directive name             | `TooltipDirective` |
| Content                    | no                |
| Permitted parent/ancestors | any               |
| Permitted children         | -                 |

## Where to use

When directive is applied to html element then hovering over the element displays tooltip above the html element.
